
#### `React`와 `Next.js` 비교

- `React` : `Library`
	- User Interface 개발할 때 사용하는 `JS Library`
	- 이미 만들어진 코드를 가져와서 우리가 원하는 형태로 사용하는 것
	- 도서관에서 필요한 책을 가져와서 읽는 것과 비슷한 개념
	- 사용 주체: 개발자
	
- `Next.js` : `Framework`
	- 사용 주체: Framework
	- 기본적인 부분은 완성된 상태에서 나머지 수정 가능한 부분은 <br/>
		개발자가 원하는 형태로 코드를 수정함.
	- `Library`는 우리가 원하는 기능들을 선택해서 사용할 수 있다. <br/>
